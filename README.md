# ng2-clockTST

## Installation

To install this library, run:

```bash
$ npm install ng2-clockTST --save
```

## Development

To generate all `*.js`, `*.js.map` and `*.d.ts` files:

```bash
$ npm run tsc
```

To lint all `*.ts` files:

```bash
$ npm run lint
```
